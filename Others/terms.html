Politique de confidentialité

SHAREFICHE s'engage à protéger la confidentialité de vos données personnelles. Cette politique de confidentialité décrit les types de données que nous collectons, la manière dont nous les utilisons et les mesures que nous mettons en œuvre pour les protéger.


1. Données collectées

Lors de votre inscription sur ShareFiche, nous pouvons être amenés à collecter les données suivantes :

  • Nom (facultatif) : Utilisé uniquement pour vous identifier personnellement.
  • Nom d'utilisateur : Nécessaire pour accéder à votre compte.
  • Mot de passe : Crypté pour assurer la sécurité de votre compte.

2. Utilisation des données

Les données que nous collectons sont principalement utilisées pour :

  • Assurer la sécurité du site : Protéger le site contre toute activité malveillante.
  • Analyser le trafic : Grâce à notre partenaire Cloudflare, nous collectons des données anonymes sur le trafic du site afin d'améliorer les performances et l'expérience utilisateur.
  • Gérer votre compte : Vous permettre de vous connecter et d'accéder aux ressources du site grâce à un token unique stocké dans un cookie.
  • Assurer la session utilisateur : Le cookie PHPSESSID permet de maintenir votre session active lors de votre navigation sur le site.
  • Améliorer l'expérience utilisateur : Nous utilisons des cookies tiers provenant de Cloudflare, mais également d'autres fournisseurs pour analyser le trafic et améliorer les performances du site. Ces cookies peuvent être déposés par les domaines suivants : ajax.googleapis.com, creativecommons.org, mirrors.creativecommons.org, jsdelivr.net, cdn.jsdelivr.net, net.cdn.cloudflare.net, cdnjs.cloudflare.com, code.jquery.com, fonts.googleapis.com, fonts.gstatic.com, cdn.jsdelivr.net, cdnjs.cloudflare.com

3. Partage des données

Nous ne partageons aucune de vos données personnelles avec des tiers, à l'exception de notre hébergeur O2switch (hébergé par Sunny) et de notre partenaire Cloudflare. Ces derniers sont soumis à des obligations contractuelles strictes en matière de protection des données.

4. Durée de conservation des données

Les données personnelles que vous nous fournissez sont conservées pendant une durée maximale d'un an à compter de leur collecte. Toutes les données seront supprimées le 01/09/2025.

5. Sécurité des données
